## Games

These games serve as examples of how to use the framework. Games under the
`pgzero` directory are designed to only work on Desktop computers running
full Pygame Zero. Games under `multi-platform` are designed to run on all
supported devices that meet the specs of a Adafruit EdgeBadge (which the
games were developer to run on). None of the games are original ideas but
developed as tributes to the legendary games I loved playing as a child.
